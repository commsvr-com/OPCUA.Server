This project provides configuration required to expose OPC UA ADI server on the Internet using CommServerUA Rel 1.00.32.

To install:

  1. Install professional licenses for CommServer and CommServer - wrapper constrain must be allowed.
  2. Copy configuration file embeddedexample/adisampleconfiguration/defaultconfig.xml to CommServer ApplicationData C:\ProgramData\CAS\CAS.CommServer
  3. Copy the EmbeddedExample folder to the C:\ProgramData\CAS\CAS.CommServerUA folder
  4. Copy CAS.CommServerUA.exe.config to the C:\Program Files (x86)\CAS\CommServerUA
